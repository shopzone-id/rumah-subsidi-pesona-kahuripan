# Rumah Subsidi Pesona Kahuripan Bogor

Website resmi informasi **Rumah Subsidi Pesona Kahuripan Bogor**.
Solusi hunian terjangkau dengan **KPR Subsidi FLPP** untuk masyarakat berpenghasilan rendah (MBR).

## Tentang Perumahan
Rumah Subsidi Pesona Kahuripan Bogor merupakan perumahan subsidi pemerintah
yang menawarkan rumah murah di Bogor dengan cicilan ringan.
Cocok untuk karyawan, pasangan muda, dan pembeli rumah pertama.

## Informasi Utama
- Lokasi: Bogor, Jawa Barat
- Program: KPR Subsidi FLPP
- Kategori: Rumah Subsidi Pemerintah
- Target: Pembeli rumah pertama
